## üí° MIGRATION GUIDE

### **From discord.py to py-cord:**
Mostly compatible, main changes:
```python
# discord.py
@bot.command()
async def hello(ctx):
    await ctx.send("Hello")

# py-cord (slash command)
@bot.slash_command()
async def hello(ctx):
    await ctx.respond("Hello")
```

### **From discord.py to nextcord:**
Almost 100% compatible, just change imports:
```python
# Change
import discord
from discord.ext import commands

# To
import nextcord as discord
from nextcord.ext import commands
```
